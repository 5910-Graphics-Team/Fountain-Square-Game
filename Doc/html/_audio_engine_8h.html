Detailed Description

This audio engine is an FMOD wrapper with provides sound file loading, 2D/3D audio playback, looping playback, FMOD Soundbank/Event playback Implements the FMOD Studio and FMOD Core API's to allow raw audio file-based implementations as well as the use and control of FMOD Studio Sound Banks.
