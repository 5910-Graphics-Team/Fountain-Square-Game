Detailed Description

A collision detection class which can be used alone or implemented by other game object classes. Use method collidesWithSphere() to check if this SphereCollider object collides with another SphereCollider Sphere-To-Sphere collision algorithm source: http://www.miguelcasillas.com/?p=9
