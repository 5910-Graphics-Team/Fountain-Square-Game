Detailed Description

A simple AABB (Axis-Aligned Bounding-Box) collision detection class Can be implemented by child class to add AABB collision detection functionality to a game object, or can be used on its own. AABB collision detection algorithm source: http://www.miguelcasillas.com/?p=30
