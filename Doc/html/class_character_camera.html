Detailed Description

A camera class that encapsulates the game user's character, processes keyboard input and calculates the corresponding Euler Angles, Vectors and Matrices for use in OpenGL Also provides collision detection capabilities between the character and other SphereColliders. source: https://learnopengl.com/Getting-started/Camera
